# 🚀 Prasnottar – Microservices Exam Management System

Prasnottar is a scalable and production-ready exam management platform built using a microservices architecture. It is designed to handle real-world academic workflows with modular, secure, and fault-tolerant system design.

---

## 🧠 Architecture Overview

The system is decomposed into multiple independent services:

- 🔐 **Authentication Service** – Handles user authentication using JWT
- 📝 **Exam Service** – Manages exams, questions, and submissions
- 📊 **Result Service** – Processes and calculates results
- 🌐 **API Gateway** – Centralized routing and request handling
- 📡 **Eureka Server** – Service discovery for dynamic communication

---

## ⚙️ Tech Stack

- **Backend:** Java, Spring Boot  
- **Microservices:** Spring Cloud (Eureka, API Gateway)  
- **Security:** JWT Authentication  
- **Database:** MySQL  
- **Communication:** REST APIs, Feign Client  
- **Build Tool:** Maven  

---

## 🔥 Key Features

- ✅ Microservices-based scalable architecture  
- ✅ Secure authentication using JWT  
- ✅ Service discovery using Eureka Server  
- ✅ Centralized routing with API Gateway  
- ✅ Inter-service communication using Feign Client  
- ✅ Modular and extensible design
